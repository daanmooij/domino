# Domino
Library to play domino

## Entities
- Board
- DominoException
- Game
- Logger
- Player
- Stock
- Tile

## Running the tests

To run all the tests, run:
```bash
make test
```

To either run only the unit or integration tests, run:
```bash
make unit-test
make integration-test
```
